  3. Create some stuff using the constructor new Stuff(name, stuff).

  5. Use the getBoxStuff() method to get all the stuff in the box.
  6. Try creating another stuff and adding it to the box using the setBoxStuff(stuff) method;

  8. Use the getBox() method to see which box is belongs by the user.
